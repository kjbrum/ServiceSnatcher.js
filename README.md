# SocialDig.js

> Query data from your social profiles with ease.


## Documentation

http://kylebrumm.com/SocialDig.js/


## Features

- 500px - photos
- Behance - projects
- CodePen - pens
- Dribbble - shots
- Flickr - photos
- Github - repos
- Google+ - posts
- Instagram - photos
- Last.fm - recent songs
- Pinterest - boards
- Spotify - playlists
- Trello - boards
- Vimeo - videos
